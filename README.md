# 🇫🇷 TEF Master — Free French Learning Platform

**Your free path to TEF Canada CLB 7**

A comprehensive, interactive study toolkit for the TEF Canada exam. No account needed. No ads. 100% free.

## Features

- **Diagnostic Test** — 44-question assessment across Grammar, Vocabulary, Reading & Listening
- **Grammar Reference** — 17 topics from A2 to B2 with practice exercises
- **Vocabulary Builder** — 200+ words across 11 themes with quiz mode
- **Reading Comprehension** — 10 TEF-style passages (A2–B2) with questions
- **Listening Practice** — 10 exercises with transcript reveal and speed control
- **Conversation Practice** — 12 real-life dialogues with fill-in-the-blank mode
- **Progress Dashboard** — XP system, badges, streaks, and study recommendations

## Tech Stack

Pure HTML/CSS/JavaScript — no frameworks, no build tools, no dependencies.

- Static multi-page site
- localStorage for progress tracking
- Responsive mobile-first design
- Deployed on Netlify with CI/CD

## Development

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/tef-master.git
cd tef-master

# Serve locally (any static server works)
npx serve .
# or
python3 -m http.server 8000
```
